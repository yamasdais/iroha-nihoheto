# アサーティヴコミュニケーション
言いにくいことを相手に伝える技法。

上手に断る、上手に主張を伝える。
相手も自分も大切にしながら、状況に応じて適切に表現する。

信頼関係を築き上げる手段
:  構築するのは大変だが、一瞬で壊れることもあるので注意が必要というニュアンスを含む

アサーティヴコミュニケーションの適用場面
: 本音を伝えることに**葛藤**が起きる場面

アサーション
: プログラミングでいうアサーション（前提条件を表明）もあるが、ここでは「自分も相手も大切にする自己表現を行うためのコミュニケーション技術」と定義される。
表明という意味で同じというだけっぽい。

### 自己表現の3パターン
* 攻撃的 I'm OK. You are not OK.
* 非主張的  I'm not OK. You are OK.
* アサーティヴ  I'm OK. You are OK.

常にアサーティヴであれば良いという訳ではない。
自分の選択に納得した上で、態度を決めれば良い

アサーティヴな自己表現とは、相手のことを大切に配慮しながら、自分の意思や感情、価値観を正直に伝える事。

アサーティヴな自己表現の例
* お客様の依頼に対して困難な気持ちを正直に伝える
* メンバの反応を恐れず自分の意見を伝えてみる
* プロジェクトの成果を自分の行動の結果として責任を取る
* 部下の優れた行動を素直に褒める
* 同僚の失敗を認め、自発的にフォローする

### 基本的な考え方
* 感情は自分のものである
* 相手とどうなりたいのかを描く
* 相手のことをきちんと聴く
* 自己責任を持つ（自分がどの様な自己表現を採るか、納得した上で）

### 効果
* 自分の意見を率直に伝える
* 我慢や恨みなどのストレスを溜め込まない
* 相手との関係

感情を伝えるのと、感情的になるのは別。
きちんと、伝えるべきことを具体的に伝える（言い方はよく考える）。
抽象的（無理・時間がない）表現は逆効果。

アサーティヴであることの心構え
1. 誠実であること
2. 率直であること
3. 対等であること
4. 自己責任を持つこと

### 準備

1.  自分と会話する
  * 自分に何が起こっているのか、客観視する
  * 自分の本音を深く見つめて受け止める
  * 相手への要望を具体的に言葉にする
  * 相手と作りたいゴール（落とし所）を描く

2.  相手と会話する
      コミュニケーションテクニックを意識する（傾聴）
  * 自分の現状を客観的に伝える
  * 相手への要望を建設的に提案する
  * 相手の考えと感情を理解しようとする（対等な立場）
  * 相手と協力して結果を作ろうとする

## ツール
### DESC法
伝えたい内容を予め４つの観点で整理する方法
まず情報を洗い出して、４つの観点に整理する
（自分の状況、相手の状況、背景の認識）

Describe
: 客観的に状況を描写する

Explain, Express
: 主観的に Desc に対する自分の気持ちを表現し、自身への影響を説明する

Suggest, Specify
: 解決策や妥協案を提案する。具体的に

Choose, Consequence
: 議案に対する肯定、否定それぞれの結果に対する行動の選択肢を示す

会話の準備の話、相手の有る事なので、順番に話せばいいという事ではない。

##### (D)共通認識を持つ
##### (E)あるべき姿を伝える
##### (S)具体的なアクションを提示する（お互いにとって最良の状態は何かをまず考える）
##### (C )有りたい姿の共有と感謝を伝える

#### 勘所
1. 時間と場所を選ぶ。相手の都合を見極める
2. 肯定的に始める
3. 事実を客観的に述べる
4. 気持ちや利害を伝える
5. 提案する
6. 選択肢を示す
